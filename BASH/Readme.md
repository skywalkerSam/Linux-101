# introduction to [Bash](<https://en.wikipedia.org/wiki/Bash_(Unix_shell)>)

## `executable` permission

```bash
chmod +x hello-world.sh
```

## run

```bash
./hello-world.sh
```

```bash
bash hello-world.sh
```

```zsh
zsh hello-world.sh
```
